# KogiQA

A modern UI automation framework that enables end-to-end testing without CSS selectors.

## Overview

![KogiQA](./assets/kogiqa.svg)

KogiQA is a powerful UI automation tool designed to simplify test automation by eliminating the need for traditional CSS selectors. This approach makes tests more maintainable, resilient to UI changes, and easier to write.

## Key Features

- **Selector-Free Automation** – Test your UI without relying on CSS or XPath selectors
- **Reliable Testing** – Reduce test fragility caused by selector changes
- **Easy to Use** – Intuitive API for writing and maintaining test cases
- **Comprehensive Coverage** – Support for end-to-end testing workflows
